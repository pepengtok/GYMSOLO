Tutorial media checklist (offline-first)
=======================================

Tempatkan file tutorial per latihan di folder ini:
- app/src/main/assets/image/tutorial/

Format nama file:
- pakai slug lowercase + underscore
- ekstensi yang didukung: .png, .jpg, .webp

Contoh:
- chest_press_mesin_hg60.png
- lat_pulldown_high_pulley_hg60.jpg
